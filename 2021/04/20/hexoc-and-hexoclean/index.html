一个关于hexo命令的小错误
2021-04-20|engineering
字数总计: 211|阅读时长: 1 分钟

hexo c 并不是 hexo clean 的简写，清理blog缓存请用 hexo clean

一个小错误

c10udlnk:如何从无到有搭建属于你的hexo博客

以上是我搭建这个个人博客时主要参考的文章

这是一个整合式教程，原作者在其中某部分参考了:

hexo c | hexo g | hexo d

意在清理缓存后生成静态文件并部署到网站

然而，在官方文档中根本没有hexo c 是 hexo clean 的简写这样一说

在实操中也可以发现，hexo c并不是一个合法的命令

(可以尝试下输入如hexo qwer等无意义的字符串，得到的反馈和截图一致

因此，在需要使用清理缓存等功能时，请使用 hexo clean
